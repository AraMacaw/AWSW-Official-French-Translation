Petit guide écrit pour tester les traductions en jeu :

- La première étape est d'installer les ModTools sur son AWSW. Pour cela, rien de plus simple. Rendez-vous sur le Steam Workshop pour AWSW, et abonnez-vous à n'importe quel mod. Cela intallera les ModTools automatiquement, vous permettant d'installer ou de créer des mods comme bon vous semble.

- Allez dans le répertoire où sont stockés les mods, c'est-à-dire, "Angels with Scaly Wings\game\mods"

- Mettez-y le dossier "fr_trad_mod" que vous trouverez à la racine du git

- Copier le dossier "frenchtl" dans le dossier "Angels with Scaly Wings\game\tl"

- Lancez le jeu, et une fois à l'écran titre, ouvrez la console (Shift+O) et tapez la commande : config.language = "french"
Lancer une partie ou une sauvegarde, puis retournez à l'écran titre, le jeu est maintenant en français
